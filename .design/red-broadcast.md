## Spacing

Base: 8px

## Do's

- Make content thumbnails the dominant visual element
- Use consistent aspect ratio for all thumbnails
- Clamp titles to exactly 2 lines with ellipsis for grid consistency
- Show metadata compactly
- Prioritize clean, scannable layouts

## Don'ts

- Overuse the accent color — reserve it for key interactive elements only
- Add borders or shadows to cards — let content sit directly on the background
- Use custom scrollbars — rely on native scrolling for performance
